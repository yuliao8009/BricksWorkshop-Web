# 卓越科技 BricksWorkshop 官方展示网站

## 🌟 关于卓越科技 (BricksWorkshop)
卓越科技（BricksWorkshop）是西北首家专业的 VEX 学苑（VEX Robotics Academy），坐落于美丽的汉中滨江路。我们秉持“专注度、游戏化学习”的教育理念，致力于为青少年提供系统、专业、且充满乐趣的机器人科技教育体验。从低龄启蒙的 VEX 123 / GO，到高阶竞技的 VEX IQ / V5，我们为孩子们铺设了一条完整的科技成长与赛事进阶之路。

## 💻 网站简介
本网站为卓越科技的官方门户与数字化名片，旨在向家长、学员以及所有科技教育爱好者全面展示我们的教育理念、课程体系与校区风采。

在视觉设计上，网站打破了传统科技教育机构沉闷的深色调，采用**极简科技风**。以大面积的全白背景为基底，辅以充满活力的“彩虹风格”色彩点缀，营造出积极向上、纯粹且富有创造力的青少年教育氛围。通过流畅的视差滚动与交互动画，为访客提供沉浸式的数字浏览体验。

## 🗺️ 核心板块导览
网站采用现代化单页平滑滚动架构，主要包含以下核心展示板块：

- **首屏印象 (Hero)**：极具视觉冲击力的机构品牌主视觉与核心宣传标语。
- **实力展现 (Statistics)**：以直观的数据呈现卓越科技的教学成果与专业底蕴。
- **教育理念 (About)**：深入解读我们的教学初衷及“游戏化学习”的独特优势。
- **课程体系 (Curriculum)**：详尽展示全年龄段的 VEX 机器人课程矩阵。
- **成长进阶 (Pathway)**：为学员量身定制的科技学习与国内外赛事晋级路线图。
- **校区风貌 (Campus & Gallery)**：全方位展示汉中滨江路校区的一流教学环境、课堂精彩瞬间与学员的高光时刻。
- **答疑解惑 (FAQ)**：贴心解答家长在机器人学习规划及课程报名中的常见问题。

## 🛠️ 技术驱动
网站不仅在设计上追求极致，在底层技术上也采用了业界前沿的现代化前端方案，确保最佳的性能与访问体验：
- **核心框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式与交互**：Tailwind CSS 驱动的响应式布局，结合 Framer Motion 打造丝滑流畅的视差滚动与交互动效。

## 🚀 运行与部署 (Quick Start)
本网站为标准的前端静态项目，本地运行与上线部署流程如下：

1. **安装依赖**：在项目根目录运行 `npm install`（或 `pnpm install`）。
2. **本地开发**：运行 `npm run dev` 启动开发服务器。
3. **打包部署**：运行 `npm run build`，将生成的 `dist` 目录托管至任意静态服务器（如 Vercel, Netlify, Nginx 等）即可完成部署。
